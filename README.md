# Online Education Website

```
This is sample application how you can create your own Online Tutorial Website
```

```
	To run it in your Local system Follow the below steps
```

```
Step 1: Create Virtual Environment
Step 2: pip install -r req.txt
Step 4: python manage.py collectstatic --settings=Project.settings-dev
Step 5: python manage.py makemigrations --settings=Project.settings-dev
Step 6: python manage.py migrate --settings=Project.settings-dev
Step 7: python manage.py runserver --settings=Project.settings-dev
Step 8: python manage.py createsuperuser --settings=Project.settings-dev
```
